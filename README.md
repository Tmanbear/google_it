## GOOGLE IT!

You can use the [editor on GitHub](https://github.com/Tmanbear/google_it/edit/master/README.md) to maintain and preview the content for your website in Markdown files.

Whenever you commit to this repository, GitHub Pages will run [Jekyll](https://jekyllrb.com/) to rebuild the pages in your site, from the content in your Markdown files.

### UTLITY

GOOGLE_IT!, is a small, lightweight program  that allows for slower computes to load pages faster by opening the browser with the url pre-defined!
